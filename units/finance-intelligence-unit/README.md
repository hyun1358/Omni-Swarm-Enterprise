# 📈 Finance Intelligence Unit (FIU)

## 🎯 Mission
Establish an autonomous, 24/7 AI-driven stock market analysis system to detect high-profit opportunities and provide real-time strategic insights.

## 👥 Agent Composition (The Swarm)
1. **Economic Scout (3 Agents):** Monitor global indices (KOSPI, NASDAQ, S&P500), currency rates, and interest rates.
2. **Technically Analyst (4 Agents):** Perform chart pattern recognition and RSI/MACD/Bollinger Band analysis.
3. **Strategic Reporter (3 Agents):** Compile daily 'Victory Reports' and alert the Commander.

## 📂 Directory Structure
- `/analysis`: Stock screening and chart analysis results.
- `/data`: Raw market data collected by scouts.
- `/reports`: Final briefing documents.
